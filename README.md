### A brief tutorial  
#### Requeriments  
* Openbox
* Tint2
* Nitrogen
* Lxappearance
* Xfce4-terminal
* Flameshot
* Plank

1. Clone repository
2. Move all openbox-files/* to .config/openbox
3. Set Wallpaper with nitrogen
4. Restart your openbox

#### Optional
`git clone https://github.com/addy-dclxvi/conky-theme-collections ~/.config/conky --depth 1`
